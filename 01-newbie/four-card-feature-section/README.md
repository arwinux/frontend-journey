## Overview

This project is a responsive landing page that presents four feature cards with a clear visual hierarchy and polished typography.

### The challenge

- View a four-card layout that highlights key product features.
- Read concise descriptions paired with icons in each card.
- See the interface adapt smoothly between mobile and desktop viewports.
- Recognize visual emphasis through colored top borders and soft shadows.
- Review the content in a clean, centered presentation.
